# SmartCar
 Python Program case about SmartCar.
 
 ```
 1. passive_buzzer.py --- 蜂鸣器
 2. button.py --- 按键启动
 3. basic_movement.py --- 基本的前后左右操作
 4. tracking --- 循迹
 5. infrad_avoid.py --- 红外壁障
 6. Avoidance --- 超声波避障
 7. ultrasonic_Servo --- 超声波避障（带舵机）
 8. ircontrol --- 红外遥控
 9. LCD1602.py --- LCD1602液晶显示
 10. WIFI_SERVO --- WIFI 视频控制
 11. led.py --- LED灯闪烁
 ```
